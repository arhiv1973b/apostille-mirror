# ТРЕБОВАНИЕ ОЧИСТКИ ИНФОРМАЦИОННЫХ СИСТЕМ МВД (EXPUNGEMENT)
**Case ID:** CASE-MACHERET-1997-2026 | **Subject:** A©tor Alexei Maceret

Настоящий узел фиксирует официальный ультиматум Министерству внутренних дел РМ об аннулировании криминальных записей.
**Основание:** Записи базируются на решении 1997 года, которое подменило факт применения пыток административным правонарушением. Поскольку абсолютный запрет пыток — это императивная норма *jus cogens*, любые производные записи в базах данных государства являются юридически ничтожными (принцип «отравленного дерева»).
