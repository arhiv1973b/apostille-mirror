# 📜 MANIFESTUL OBSERVATORULUI
**Obiectiv:** Ședința Curții de Apel Chișinău din 09.04.2026

## 1. PRINCIPIUL SUPERPOZIȚIEI
Până la 09.04, sistemul judiciar este în stare de superpoziție:
* **Starea A (Reabilitare):** Recunoașterea normei Jus Cogens (tortura din 1998).
* **Starea B (Colaps):** Utilizarea barierelor procedurale. Aceasta confirmă moartea dreptului internațional.

## 2. EFECTUL DOPPLER ÎN DREPT
Data de 11.02.2021 (Apostila) este momentul în care adevărul din 1998 a ajuns în câmpul juridic. Orice refuz este o "falsificare științifică".
