## KEY FINDINGS

### 1. CPT Standards Are Mandatory

- Moldova ratified European Convention for Prevention of Torture
- CPT recommendations are binding
- State obligated to implement

### 2. Systemic Non-Compliance

| Area                  | Problem                  | Status       |
| --------------------- | ------------------------ | ------------ |
| Police custody        | Ill-treatment            | Not resolved |
| Detention conditions  | Below standards          | Not resolved |
| Prisons               | Overcrowding, conditions | Not resolved |
| Mental health         | Institutional abuse      | Not resolved |
| Legislative framework | Incomplete               | Not resolved |
| Training              | Insufficient             | Not resolved |

### 3. Government Awareness

- Report acknowledges problems exist
- Recommendations not implemented
- Pattern of inaction documented

## RELEVANCE TO CURRENT CASE

1. ✅ Moldova knows about CPT standards

3. ✅ Moldova fails to act

### Legal Implications:

- **Continuing breach** - established
