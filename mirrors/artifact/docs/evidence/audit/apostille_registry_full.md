# ПОЛНЫЙ РЕЕСТР АПОСТИЛЕЙ (МАТЕМАТИЧЕСКАЯ БАЗА ДОКАЗАТЕЛЬСТВ)
**Case ID:** CASE-MACHERET-1997-2026 | **Subject:** A©tor

В данный узел интегрирован полный реестр из 90 верифицированных государственных апостилей.
Реестр является фундаментальной доказательной базой, подтверждающей системные аномалии (14.4% Void Rate), паттерны анонимизации подписей должностных лиц ("indescifrabilă") и масштаб институциональной имитации правосудия (justice simulation), активирующей юрисдикцию Международного уголовного суда (Статья 17 Римского статута).
